# terraform-to-aws

## Beberapa command terraform yang akan sering digunakan
```
terraform apply (untuk apply terraform anda)
terraform plan (untuk melihat apa saja yang apps yang akan diapply)
terraform fmt (untuk memformat code anda agar enak dipandang)
terraform console (untuk masuk pada console)
terraform destroy (untuk menghapus apps)
terraform show (untuk melihat apa yang sudah kalian apply)
terraform show state{nama state} (sama halnya seperti show namun lebih spesifik)
terraform init (untuk initalisasi/menginstall terraform dan providernya)
```

## Adapun tambahannya seperti
```
-replace (untuk mengganti apps)
-plan (untuk melihat apa yang akan dilakukan sebelum menjalankan perintah utama)
-auto-approve (agar langsung menyetujui persyaratan)
-refresh-only (hanya untuk merefresh dan bukan mengganti)
```
